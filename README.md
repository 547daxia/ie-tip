# ie-tip
当用户使用IE内核浏览器时提示他换其他浏览器



```bash
npm i ie-tip
```



```js
import ieTip from 'ie-tip'
Vue.use(ieTip)
```



```html
<!-- 放到任意页面当检测到浏览器是IE内核时 会弹出提醒 -->
<template>
  <div>
    <ie-tip></ie-tip>
  </div>
</template>
```
